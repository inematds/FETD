# Módulo 07: Slack e Comunicação Assíncrona

**Trilha:** Talento Emergente
**Duração:** 45 minutos
**Objetivo:** Dominar ferramentas de comunicação profissional

---

## Introdução

Trabalho remoto exige comunicação eficaz. Slack é padrão em 80% das startups brasileiras.

## Conteúdo Principal

### 1. Slack Profissional

**Canais essenciais:**
- #geral: announcements
- #projetos: por projeto
- #random: casual
- #vendas, #tech, etc.

**Etiqueta:**
- Use threads (não polua canal)
- @channel apenas urgente
- Status claro (disponível/ocupado/foco)
- Reaja com emoji (✅ = visto)

### 2. Comunicação Assíncrona

**Mensagem eficaz:**
```
❌ "Oi, tudo bem? Podemos conversar?"
✅ "Oi [Nome]! Sobre projeto X:

Contexto: [1 linha]
Pergunta: [específica]
Urgência: [quando preciso]
Arquivos: [anexos]

Pode responder quando conseguir!"
```

**Elementos:**
1. Contexto claro
2. Pergunta específica
3. Deadline
4. Tudo necessário (não ping-pong)

### 3. Slack Bots

**Automações:**
- Daily standup bot
- Reminder automático
- Integração GitHub/Trello
- Notificações Make

### 4. Gestão de Notificações

**Configurar:**
- DND (Do Not Disturb) horários
- Keywords importantes
- Silenciar canais low-priority
- Custom notifications por canal

## Exercício

1. Configure Slack pessoal
2. Crie 5 canais organizados
3. Integre 1 bot/automação
